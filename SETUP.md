## üöÄ Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
npm install
```

### 2. Register Azure AD App
1. Go to https://portal.azure.com
2. Azure Active Directory ‚Üí App registrations ‚Üí New registration
3. Name: "SharePoint MCP"
4. Redirect URI: Web ‚Üí `http://localhost:3000/callback`
5. Add API permissions: Sites.Read.All, Files.Read.All, offline_access
6. Grant admin consent
7. Copy Client ID and Tenant ID

### 3. Configure Claude Desktop

Edit config file:
- **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
- **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`

Add:
```json
{
  "mcpServers": {
    "sharepoint": {
      "command": "node",
      "args": ["C:\\Users\\ofek\\Downloads\\gitRepos\\sharepoint-mcp\\index.js"]
    }
  }
}
```

**Important**: Use the full absolute path to index.js!

### 4. Restart Claude Desktop

Completely quit and restart Claude Desktop.

### 5. Test It

In Claude, say:
```
Authenticate with SharePoint using:
- Client ID: [paste-your-client-id]
- Tenant ID: [paste-your-tenant-id]
```

Then:
```
Set my SharePoint site to: https://yourtenant.sharepoint.com/sites/yoursite
```

Finally:
```
Search for files containing "test" in my SharePoint
```

## üÜò Common Issues

**"MCP server not found"**
- Verify path in Claude config is absolute
- Restart Claude Desktop completely
- Check Node.js is in PATH

**"Authentication failed"**
- Verify Client ID and Tenant ID are correct
- Check redirect URI matches exactly
- Ensure permissions are granted

**"Permission denied"**
- Click "Grant admin consent" in Azure
- Verify your account has SharePoint access
